# Math

## Methods
```javascript
// Constants
Math.E;               // Constante de Euler (2.718281828459045).
Math.PI;              // Constante PI (3.141592653589793).
Math.LN2;             // Logaritmo natural de 2 (0.6931471805599453).
Math.LN10;            // Logaritmo natural de 10 (2.302585092994046).
Math.LOG2E;           // Logaritmo base 2 de E (1.4426950408889634).
Math.LOG10E;          // Logaritmo base 10 de E (0.4342944819032518).
Math.SQRT2;           // Raíz cuadrada de 2 (1.4142135623730951).
Math.SQRT1_2;         // Raíz cuadrada de 1/2 (0.7071067811865476).

// Rounding constants
Math.ROUND_UP;        // Redondeo hacia arriba.
Math.ROUND_DOWN;      // Redondeo hacia abajo.
Math.ROUND_HALF_UP;   // Redondeo hacia el valor más cercano.
Math.ROUND_HALF_DOWN; // Redondeo hacia el valor más cercano, si está a la mitad redondea hacia abajo.
Math.ROUND_HALF_EVEN; // Redondeo hacia el valor más cercano, si está a la mitad redondea hacia el valor par más cercano.
Math.ROUND_CEIL;      // Redondeo hacia arriba.
Math.ROUND_FLOOR;     // Redondeo hacia abajo.

// Arithmetic functions
Math.abs(-5);         // Valor absoluto (5).
MAth.pow(2, 3);       // Potencia (8).
Math.sqrt(25);        // Raíz cuadrada (5).
Math.sign(-5);        // Devuelve -1 si el número es negativo, 1 si es positivo, 0 si es cero y NaN si no es un número.
Math.random();        // Número aleatorio entre 0 y 1.
Math.max(2, 3, 5);    // Máximo (5).
Math.min(2, 3, 5);    // Mínimo (2).

// Rounding functions
Math.round(5.5);      // Redondeo hacia el valor más cercano (6).
Math.ceil(5.1);       // Redondeo hacia arriba (6).
Math.floor(5.9);      // Redondeo hacia abajo (5).
Math.trunc(5.9);      // Elimina los decimales (5).

// Trigonometric functions
Math.sin(1);          // Seno (0.8414709848078965).
Math.cos(1);          // Coseno (0.5403023058681398).
Math.tan(1);          // Tangente (1.5574077246549023).
Math.asin(1);         // Arco seno (1.5707963267948966).
Math.acos(1);         // Arco coseno (0).
Math.atan(1);         // Arco tangente (0.7853981633974483).
Math.atan2(1, 1);     // Arco tangente de y/x (0.7853981633974483).
```
