# Number

## Methods
```javascript
let num = 123.456;

Number.MAX_VALUE;          // Número más grande representable.
Number.MIN_VALUE;          // Número más pequeño representable.
Number.POSITIVE_INFINITY;  // Infinito positivo.
Number.NEGATIVE_INFINITY;  // Infinito negativo.
Number.NaN;                // Not a Number.

num.toString();            // Número como string.
num.toFixed(2);            // Número con 2 decimales.
num.toExponential(2);      // Notación exponencial con 2 decimales.
num.valueOf();             // Valor primitivo del objeto.

Number.isFinite(num);      // Devuelve true si el número es finito.
Number.isInteger(num);     // Devuelve true si el número es entero.
Number.isNaN(num);         // Devuelve true si el número es NaN.

parseInt("123");           // Convierte un string a entero.
```
